# SkillCraft Technologies – Web Development Internship

## Task 02: Stopwatch Web Application

This project is an interactive and user-friendly stopwatch web application created as part of the SkillCraft Technologies Web Development Internship.

---

## 🔧 Technologies Used
- HTML5  
- CSS3  
- JavaScript  

---

## ✨ Features
- Start, pause, and reset functionality  
- Accurate time tracking using JavaScript  
- Lap time recording and display  
- Circular stopwatch UI design  
- Animated background and modern visual effects  
- Responsive layout for different screen sizes  

---

## 📂 Project Structure
SCT_WD_02/
│── index.html
│── style.css
│── script.js

---

## 📌 Description
The stopwatch web application allows users to accurately measure and record time intervals. It includes lap tracking functionality and a visually engaging circular interface. The project demonstrates practical JavaScript logic, DOM manipulation, and modern CSS styling techniques without using any external libraries or frameworks.
